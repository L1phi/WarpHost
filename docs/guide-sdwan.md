# SD-WAN 教程

SD-WAN 模式用于没有公网 IP、NAT 环境、宿舍网、校园网或朋友 P2P 联机。

WarpHost 使用内置的 EasyTier 创建虚拟局域网。

## 准备工作

安装包版本会内置：

```text
easytier-core.exe
```

开发环境必须手动放到：

```text
resources/easytier-core.exe
```

## 操作步骤

1. 打开 WarpHost。
2. 进入 **本地机房**。
3. 在 SD-WAN 区域输入网络名称。
4. 输入至少 8 位网络密钥。
5. 点击连接。
6. 等待虚拟 IP 出现。

## 多人联机

所有玩家必须使用相同的：

- 网络名称
- 网络密钥

开服者启动游戏后，把虚拟 IP 和游戏端口发给其他玩家：

```text
10.x.x.x:25565
```

## 常见问题

### EasyTier 启动失败

检查安装包是否完整，杀毒软件是否拦截，网络名称和密钥是否合法。

### 没有虚拟 IP

等待几秒，检查本机防火墙，确认其他机器使用同一个网络名称和密钥。

### 能看到虚拟 IP 但连不上游戏

确认游戏容器已启动，端口正确，玩家连接的是开服者虚拟 IP。
