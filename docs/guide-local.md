# 本地机房教程

本地机房适合本机或家宽网络拥有公网 IPv6 的场景。

## 准备工作

确认你有：

- Windows 系统
- Docker Desktop 已安装并启动
- 本机拥有公网 IPv6
- 路由器允许 IPv6 入站

## 操作步骤

1. 启动 Docker Desktop。
2. 打开 WarpHost。
3. 进入 **本地机房**。
4. 选择游戏蓝图。
5. 点击本地部署。
6. 等待日志输出 IPv6 连接地址。

## IPv6 地址格式

IPv6 连接地址通常长这样：

```text
[240e:xxxx:xxxx::1]:25565
```

方括号不能省略。

## 防火墙

WarpHost 会在部署时尝试打开对应 TCP 端口。如果提示权限不足，请以管理员身份运行 WarpHost。

## 常见问题

### 未检测到公网 IPv6

如果系统只有 `fe80::` 开头的地址，那通常不是公网 IPv6。请检查宽带、路由器和系统网络配置。

### 玩家连不上

检查玩家网络是否支持 IPv6，路由器是否允许入站，Windows 防火墙是否开放端口。
