# 云端跃迁教程

云端跃迁适合你已经拥有 VPS 或云服务器的场景。

## 准备工作

确认你有：

- 服务器公网 IP
- root 密码或具有 Docker 权限的账号
- 云厂商安全组已放行游戏端口
- 服务器可以访问 Docker 镜像仓库

## 操作步骤

1. 打开 WarpHost。
2. 进入 **云端跃迁**。
3. 输入服务器 IP。
4. 输入 root 密码。
5. 选择游戏蓝图。
6. 点击 **一键跃迁**。
7. 等待日志显示部署成功和连接地址。

## 端口提醒

如果蓝图端口是 `25565`，云安全组也必须放行 `25565/TCP`。

部署成功但玩家连不上时，优先检查安全组和系统防火墙。

## 常见问题

### SSH 连接失败

检查 IP、密码、22 端口、安全组、服务器是否允许密码登录。

### Docker 不可用

WarpHost 会尝试检查和启动 Docker。如果失败，请登录服务器手动执行：

```bash
docker --version
docker ps
```

### 镜像拉取失败

检查服务器网络、镜像名、Docker Hub 可访问性。
